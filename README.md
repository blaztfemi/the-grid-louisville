# The Grid — Louisville Career Intelligence

An open-source career intelligence network for Louisville young people, parents, and families navigating alternative pathways to the four-year college track.

Built by Work Architecture. Community powered.

## What It Is

The Grid maps Louisville's hidden economy — trades, emerging tech, entrepreneurship, certifications, and apprenticeships that most families never hear about. It connects people to real organizations, real contacts, and real information.

## Structure

```
public/           The deployed static site (index.html)
research/         Internal research library (pathway nodes, prompt library)
```

## Deployment

Static site deployed on Vercel. Auto-deploys on push to main.

## Tech

Single-file HTML with vanilla CSS and JavaScript. No framework dependencies. No build step required.

## Contributing

Submit intel through the "Share What You Know" form on the site. All community submissions feed the map.
